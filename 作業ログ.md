### Amazon RDS
```bash
# 必要に応じて apt や brew で direnv を入れる
direnv allow .
docker run -it --rm \
    postgres:17.4 psql \
    -h $(terraform output rds_endpoint) \
    -U $TF_VAR_rds_username \
    -d postgres
```
